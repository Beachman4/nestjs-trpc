# Context

Currently, the only way to extend the default tRPC context is by passing a context function to the module options, specifically the `` option:
```typescript filename=app.module.ts
import { Module } from '@nestjs/common';
import { TRPCModule } from 'nestjs-trpc';

@Module({
  imports: [
    TRPCModule.forRoot({
      createContext: (opts) => {

      },
    }),
  ],
})
export class AppModule {}
```
